# Dormant Account Detector

An IAM (Identity and Access Management) security tool that scans user accounts and flags any that have been inactive for 30 or more days, so they can be reviewed or disabled before they become a security risk.

**Course:** Security Architecture.
**Group:** Epsilon.


This project has two versions:
1. **Terminal Version**: It displays the report directly in the console.
2. **GUI Version**: It displays the report inside a simple desktop window.


# GUI Version: dormant_account_detector_gui_simple.py

*What it does:*
Same detection logic as the terminal version, but displayed inside a graphical window instead of the console and is built using **tkinter**, which comes built into Python (no installation needed).
